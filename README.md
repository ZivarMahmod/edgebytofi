# EDGE by Tofi

Statisk webbplats för EDGE by Tofi — barbershop i Linköping.
Ren HTML/CSS/JS, inga byggsteg. Deployas på Cloudflare Pages.

## Kör lokalt
```
python -m http.server 8000
```
Öppna http://localhost:8000

## Deploy (Cloudflare Pages)
Inget byggkommando. Output-katalog: `/` (roten).

Bokning: [bokadirekt.se/places/edge-by-tofi-136651](https://www.bokadirekt.se/places/edge-by-tofi-136651)
